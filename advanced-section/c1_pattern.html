একটি Pattern তৈরি করতে হলে print() কে Line by Line ব্যবহার করতে হয়!

Python কোড এক লাইনের পর আরেক লাইন উপরে থেকে নিচে রান হয়।

আমরা একাধিক print() ফাংশন ব্যবহার করে একাধিক মেসেজ দেখাতে পারি।

▶ Line 1 runs first → Output 1
▶ Line 2 runs next → Output 2
▶ Line 3 runs after → Output 3
উদাহরণ
একাধিক print() ব্যবহার:
📄 example.py
1print('🌻 Morning Dharma!')
2print('🙋🏽 Evening Sonny!')

এটির output হবে:

OUTPUT
$ python example.py
🌻 Morning Dharma!
🙋🏽 Evening Sonny!
