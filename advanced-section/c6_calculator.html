🧮 পাইথন ক্যালকুলেটর

দুটি জাদুকরী ফাংশন বানিয়ে যোগ-বিয়োগ করো এক পলকেই!

▸ INTRO

Calculator কী?

সহজ কথায়, "যে যন্ত্রের সাহায্যে আমরা খুব দ্রুত যোগ, বিয়োগ, গুণ, ভাগের মতো গণিতের নানা হিসাব করে ফেলতে পারি, তাকেই Calculator বলে।"

তোমাদের জ্যামিতি বক্সে বা আব্বু-আম্মুর স্মার্টফোনে তোমরা অবশ্যই ক্যালকুলেটর দেখেছ। আজ আমরা কোনো সাধারণ ক্যালকুলেটর নয়, বরং পাইথনের ফাংশন দিয়ে আমাদের নিজেদের একটি স্মার্ট ডিজিটাল ক্যালকুলেটর তৈরি করব!

▸ MISSION BRIEF

নির্দেশনা — তোমার মিশন

🤖 মিশন কন্ট্রোল থেকে বার্তা

মনে করো, গণিত ক্লাসে স্যার তোমাকে অনেক বড় বড় দুটি সংখ্যার যোগ এবং বিয়োগ করতে দিয়েছেন। তুমি বুদ্ধি করে বাড়ি এসে পাইথনে দুটি জাদুকরী ফাংশন বা রোবট বানিয়ে রাখলে—একটি যোগ করার জন্য, আরেকটি বিয়োগ করার জন্য।

তুমি যখনই যেকোনো দুটি সংখ্যা ওই ফাংশনে দেবে, সে পলকের মধ্যে তোমাকে সঠিক উত্তরটি বের করে দেবে! চলো, কোডটি লিখে ফেলা যাক।

💡 হিন্টস (Hints)

  • যোগের ফাংশন: দুটি সংখ্যা যোগ করার জন্য একটি ফাংশন তৈরি করো: def যোগ_করো(a, b): এবং ভেতরে return a + b লিখে দাও।
  • বিয়োগের ফাংশন: বিয়োগ করার জন্য আরেকটি ফাংশন তৈরি করো: def বিয়োগ_করো(a, b): এবং ভেতরে return a - b লিখে দাও।
  • প্যারামিটার ব্যবহার: ফাংশন ডাকার সময় ব্র্যাকেটের ভেতর সংখ্যা দুটি পাঠিয়ে দিতে হবে।

# ১. যোগ করার জন্য ফাংশন তৈরি (মান ফেরত দেবে) def jog_koro(a, b): return a + b # ২. বিয়োগ করার জন্য ফাংশন তৈরি (মান ফেরত দেবে) def biyoj_koro(a, b): return a - b print("--- 🧮 আমার পাইথন ক্যালকুলেটর ---") # ৩. ফাংশন কল করে হিসাব বের করা jog_fol = jog_koro(45, 15) print("➕ সংখ্যা দুটির যোগফল:", jog_fol) biyoj_fol = biyoj_koro(100, 35) print("➖ সংখ্যা দুটির বিয়োগফল:", biyoj_fol) >>> --- 🧮 আমার পাইথন ক্যালকুলেটর --- >>> ➕ সংখ্যা দুটির যোগফল: 60 >>> ➖ সংখ্যা দুটির বিয়োগফল: 65
