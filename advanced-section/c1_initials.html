📝 Programming-এ comments

Programming-এ comments খুবই গুরুত্বপূর্ণ কারণ এগুলো কোড কী কাজ করছে তা ব্যাখ্যা করতে সাহায্য করে।

এছাড়াও program-এর কিছু অংশ সাময়িকভাবে বন্ধ রাখতেও comments ব্যবহার করা হয়।

Program রান হলে comments উপেক্ষা করা হয়।

Python-এ # hashtag symbol ব্যবহার করে comment লেখা হয়:

# Printing out a message
print('Hi')

প্রথম লাইনে একটি comment লেখা হয়েছে।
তাই # এর ডান পাশে যা আছে তা Python ignore করবে।
Program পরের লাইনে যাবে এবং output হবে:

Hi

Python শুধুমাত্র একই লাইনে # এর পরের অংশ ignore করে।
যেমন:

print('Hi')   # I'm a comment, too!

এখানে "Hi" এখনও print হবে কারণ comment-এর আগের কোড execute হয়।

Hi
# ignored 🚫💬 Python sees only code → comments are invisible
