⚡ Python
hypotenuse.py 

# ১. user-এর কাছ থেকে দুটি বাহুর দৈর্ঘ্য input নেওয়া
a = int(input('a বাহুর দৈর্ঘ্য লিখো: '))
b = int(input('b বাহুর দৈর্ঘ্য লিখো: '))

# ২. Pythagorean theorem দিয়ে hypotenuse (c) বের করা
c = (a ** 2 + b ** 2) ** 0.5

# ৩. ফলাফল প্রিন্ট করা
print("Hypotenuse (c) =", c)
💻 Terminal Output:
{'>'} a বাহুর দৈর্ঘ্য লিখো: 3
{'>'} b বাহুর দৈর্ঘ্য লিখো: 4
Hypotenuse (c) = 5.0
✨ a=3, b=4 হলে c=5 (এটাই পিথাগোরাসের বিখ্যাত ৩-৪-৫ ত্রিভুজ!)
