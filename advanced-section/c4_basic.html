Level 4: Chocolate Decrementer

While Loop (শর্তের লুপ) :

While Loop একটু অন্যরকম। এটা ততক্ষণ পর্যন্ত চলতেই থাকে, যতক্ষণ না একটা নির্দিষ্ট শর্ত (Condition) সত্যি থাকে। শর্ত ভুল হলেই লুপ বন্ধ!

while_chocolate.py
>>> chocolate = 5 # তোমার কাছে ৫টি চকলেট আছে
>>> while chocolate > 0: # যতক্ষণ চকলেট ০ থেকে বেশি থাকবে, লুপ চলবে
... print("আমি একটি চকলেট খেলাম।")
... chocolate = chocolate - 1 # একটা চকলেট কমে গেল!
>>> print("সব চকলেট শেষ!")
চকলেট বয়াম লুপ (Chocolate Jar) 🍬
বয়ামে চকলেট: 5 | শর্ত (chocolate > 0): True

🔑 Solution (সলিউশন কোড)
▼

লুপের মিশন সম্পন্ন করার জন্য অফিশিয়াল সলিউশন স্ক্রিপ্ট নিচে দেওয়া হলো। এই কোডটি ভালো করে স্টাডি করো:

solution.py 
# Print message 5 times
for i in range(5):
    print("I love Python")

# Print 1 to 5 using while loop
count = 1
while count <= 5:
    print(count)
    count += 1

# Nested loop
for i in range(2):
    for j in range(3):
        print("🔥")

# Break example
for i in range(5):
    if i == 3:
        break
    print(i)

# Continue example
for i in range(5):
    if i == 2:
        continue
    print(i)
