Level 05 · Python Data Types

🧬 Data Types

পাইথনের ৪টি প্রধান ডেটা টাইপ — int, float, str আর bool — চলো একসাথে চিনে নিই!

🚀 শুরু করো

📦 ৪টি ডেটা টাইপ

🔢
INTEGER

int হলো পূর্ণ সংখ্যা। কোনো ডেসিমাল পয়েন্ট থাকে না।

year = 2023 age = 32
🌊
FLOAT

float হলো দশমিক সংখ্যা — যেমন দাম বা পাইয়ের মান।

pi = 3.14159 meal_cost = 12.99
💬
STRING

str দিয়ে টেক্সট রাখা হয়। ' ' বা " "-এর ভেতরে লিখতে হয়।

message = "good nite" username = '@snoopdogg'
⚡
BOOLEAN

bool-এ থাকে শুধু দুটো মান — True অথবা False।

late_to_class = False cranky = True
💡 প্রথম চ্যাপ্টারে আমরা print() ব্যবহার করার সময়ই string ডেটা টাইপ ব্যবহার করেছিলাম — মনে আছে?
